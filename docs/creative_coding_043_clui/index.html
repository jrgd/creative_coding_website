Hello to you too. Yay World.
This is a creative coding experiment aimed at extending the interface, the interaction templates and the various controls for the apps.
In terms of the course specifically, it's a good introduction to what a shell does and how it works
and the various complexities (and opportunities for learning new things) it hides.

Some references:
  • SRAM, on CPC
  • CLUI
  • Personnal research into code editors
At the moment only the following commands have been developed:
  • list: with arguments, hiting a php script server_commands.php
    it takes presently only one argument: what - either 'files' or 'images'.
  • clear: to clear the console.
The open command has been prototyped but is not functionnal.
Commands to be developped for version 0.1:
  • open - for images, data, movies, code, etc.
  • palette - to control/display specific palette (gui).
  • colour - for colour manipulation.
  • import - tbc.
  • export - destination formats: svg, image, movie.
  • get_color - at specific point x,y.
Features to be developped for version 0.2:
  • persistent command history - store command history in local storage.
  • download - for images, data, movies, code, etc.
  • draw commands - scripts with variables.
